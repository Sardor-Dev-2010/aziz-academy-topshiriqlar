# 📝 To-Do App — Vazifalar Ro'yxati Loyihasi

Ushbu loyiha HTML, CSS va JavaScript texnologiyalari yordamida yaratilgan oddiy va interaktiv vazifalar menejeri (To-Do veb-ilovasi) hisoblanadi. Loyihada foydalanuvchi interfeysi (UI maketi) talablariga to'liq rioya qilingan.

## ✨ Imkoniyatlari
* **Vazifa qo'shish:** Matn kiritish maydoniga yozib, "Qo'shish" tugmasini yoki klaviaturadagi `Enter`ni bosish orqali yangi vazifalar qo'shish mumkin.
* **Xatoliklardan himoya:** Foydalanuvchi bo'sh qiymat kiritganda dastur ogohlantirish xabarini beradi va bo'sh element qo'shilmaydi.
* **Vazifalarni o'chirish:** Har bir qo'shilgan vazifa yonida alohida "O'chirish" tugmasi yaratiladi va u orqali vazifani ro'yxatdan olib tashlash mumkin.
* **Zamonaviy va Responsive dizayn:** Dastur interfeysi ham kompyuter, ham mobil qurilmalar ekranlariga moslashuvchan qilib ishlangan.

## 🛠️ Loyiha Strukturasi va UI Maketi
Loyiha topshiriqdagi quyidagi asosiy UI komponentlarini o'z ichiga oladi:
* `<input id="taskInput">` — Vazifa kiritish maydoni.
* `<button id="addBtn">` — Vazifani ro'yxatga qo'shuvchi boshqaruv tugmasi.
* `<ul id="list">` — Dinamik ravishda to'ladigan bo'sh boshlang'ich ro'yxat.

## 🚀 Ishga tushirish yo'riqnomasi
1. Loyihaga tegishli barcha fayllarni (`index.html`, `style.css`, `script.js`, `README.md`) bitta papkaga joylashtiring.
2. `index.html` faylini istalgan zamonaviy brauzerda (Chrome, Firefox, Safari, Edge) ikki marta bosib oching.
3. Veb-ilovadan cheksiz foydalanishingiz mumkin.
